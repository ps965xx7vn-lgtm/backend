# Инструкция по CodeHS: как создать аккаунт и писать код 🚀

CodeHS — это отличная онлайн-платформа, которая позволяет писать код на Python прямо в браузере, ничего не устанавливая на свой компьютер. В наших первых уроках мы будем использовать именно её.

Вот простая пошаговая инструкция, как начать работу.

## Шаг 1: Регистрация

1. Перейди на сайт [CodeHS.com](https://codehs.com/).
2. В правом верхнем углу нажми кнопку **Sign Up** (Зарегистрироваться).
3. Платформа предложит выбрать роль. Обязательно выбери **Student** (Студент).
4. Если у тебя есть Google-аккаунт, быстрее всего нажать **Continue with Google**. Также можно ввести свою почту (Email) и придумать пароль.

> 💡 **Что делать, если не удаётся создать аккаунт?**
> На данный момент CodeHS работает в РФ и свободно принимает почтовые ящики. Однако, если что-то не работает — используй [прямую ссылку на Песочницу (Sandbox)](https://codehs.com/sandbox). Она позволяет временно писать код без регистрации.

## Шаг 2: Создание проекта (Python)

После регистрации ты попадёшь на главную страницу (Dashboard).
Чтобы начать писать код, нам нужно создать свой первый "проект". В CodeHS это называется **Sandbox program** (Песочница).

1. Найди в левом или верхнем меню раздел **Sandbox** (Песочница) и перейди туда.
2. Нажми большую кнопку **+ New Program** (Новая программа) в правом верхнем углу.
3. Появится окно выбора языка программирования. Найди и выбери **Python 3**.
4. Придумай название проекту (например, `RocketLaunch`) и нажми **Create Program** (Создать).

## Шаг 3: Где писать код?

Перед тобой откроется редактор. Он состоит из трёх главных блоков:
- **Слева** находятся файлы твоего проекта. По умолчанию ты работаешь в файле `main.py`.
- **По центру** находится сам **текстовый редактор**. Сюда ты будешь писать свой код!
- **Справа** — чёрное окно **Терминала** (или консоль). Здесь будет отображаться результат работы твоей программы.

## Шаг 4: Запуск программы

1. Напиши пару строк кода в центральном окне (например, `print('Привет, Центр Управления!')`).
2. Найди яркую зелёную кнопку **▶ Run** (Запустить) в правом верхнем углу и нажми её.
3. В правом чёрном окне должен появиться результат!

## Шаг 5: Поделись кодом 🔗

Когда ты выполнишь задание, тебе нужно будет отправить код на проверку нашим ревьюерам.

1. Найди кнопку **Share** (Поделиться) в правом верхнем углу интерфейса CodeHS.
2. В появившемся окне скопируй ссылку (она должна выглядеть примерно так: `https://codehs.com/sandbox/id/project-name`).
3. Убедись, что доступ открыт для просмотра.
4. Вставь эту ссылку в поле отправки задания в нашем курсе.

🎉 Вот и всё! Теперь ты готов покорять космос вместе с Python. Возвращайся к уроку и приступай к первому заданию!
