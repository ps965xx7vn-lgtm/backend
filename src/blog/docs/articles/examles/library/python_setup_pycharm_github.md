# Полная подготовка к разработке на Python с PyCharm и GitHub Desktop

Вы решили разрабатывать на `Python` и хотите настроить своё рабочее окружение? Отличный выбор! В этой статье мы пошагово расскажем, как подготовить всё необходимое для комфортной работы, используя [PyCharm](https://www.jetbrains.com/pycharm/download/?section=windows) и [GitHub Desktop](https://github.com/apps/desktop). Всё будет просто и понятно, обещаем! 😎

## Шаг 1: Установка Python 🐍
Перед тем как начать, убедитесь, что у вас установлен `Python`. Если он ещё не установлен, не переживайте, вот как это сделать:

1. Перейдите на официальный сайт `Python`: [python.org](https://www.python.org/)

2. Скачайте последнюю версию для вашей операционной системы.

3. Установите `Python`, не забыв отметить галочку "`Add Python to PATH`" в процессе установки.

4. Проверьте установку в командной строке, введя команду:

```bash
python --version
```
Если версия `Python` отобразилась, значит, всё прошло успешно! 🎉

## Шаг 2: Установка Git 🖥️
[Git](https://git-scm.com/downloads) необходим для работы с `GitHub`, поэтому убедитесь, что он установлен на вашем компьютере.

1. Перейдите на сайт [Git](https://git-scm.com/downloads) и скачайте установщик для вашей операционной системы.

2. Запустите установку и следуйте инструкциям, оставив параметры по умолчанию.

3. После установки откройте командную строку и введите:

```bash
git --version
```
Если версия `Git` отобразилась, значит, он успешно установлен.

## Шаг 3: Установка PyCharm 🧑‍💻
[PyCharm](https://www.jetbrains.com/pycharm/download/?section=windows) — это один из самых популярных `IDE` для `Python`, который делает разработку удобной и эффективной.

Перейдите на сайт [PyCharm](https://www.jetbrains.com/pycharm/download/?section=windows) и скачайте бесплатную версию `Community`.
Установите [PyCharm](https://www.jetbrains.com/pycharm/download/?section=windows), следуя инструкциям на экране.
После установки откройте [PyCharm](https://www.jetbrains.com/pycharm/download/?section=windows) и настройте рабочую среду:
Когда откроется [PyCharm](https://www.jetbrains.com/pycharm/download/?section=windows), выберите "`New Project`".
Убедитесь, что выбран правильный интерпретатор `Python` (если `Python` установлен правильно, `PyCharm` автоматически его подхватит).
Назовите проект и выберите папку, где хотите его хранить.
Теперь у вас есть готовая среда для разработки на `Python`! 🚀

## Шаг 4: Установка GitHub Desktop 💻
`GitHub` — это одна из самых популярных платформ для хранения кода и совместной работы. Для удобства работы с `GitHub` мы установим [GitHub Desktop](https://github.com/apps/desktop).

Перейдите на сайт [GitHub Desktop](https://github.com/apps/desktop) и скачайте приложение.
Установите GitHub Desktop, следуя инструкциям.
Откройте [GitHub Desktop](https://github.com/apps/desktop) и войдите в свой аккаунт `GitHub` (если у вас его нет, создайте его).
Теперь у вас есть удобный инструмент для работы с репозиториями на `GitHub` прямо с вашего компьютера.
## Шаг 5: Создание репозитория на GitHub 🌍
В [GitHub Desktop](https://github.com/apps/desktop) нажмите на кнопку "`Create a New Repository`".
Укажите имя репозитория и выберите папку для хранения проекта.
Нажмите "`Create Repository`".
Теперь у вас есть репозиторий на `GitHub` для вашего проекта, и вы можете начать работать с кодом, отслеживать изменения и делиться ими с другими.

## Шаг 6: Подключение PyCharm с GitHub Desktop 🔗
Откройте [PyCharm](https://www.jetbrains.com/pycharm/download/?section=windows) и создайте новый `Python-проект` (или откройте существующий).
Перейдите в "`VCS`" → "`Checkout from Version Control`" → "`Git`".
Введите `URL` вашего репозитория `GitHub` и выберите папку, куда хотите его клонировать.
Нажмите "`Clone`", и `PyCharm` подключится к вашему репозиторию на `GitHub`.
`Теперь вы можете работать с проектом в PyCharm и синхронизировать изменения через GitHub Desktop. Это позволит вам легко контролировать версии кода и работать над проектом с другими людьми.
`
## Шаг 7: Начало работы с проектом 👨‍💻
Теперь, когда у вас настроено всё необходимое, можно приступать к программированию!

В [PyCharm](https://www.jetbrains.com/pycharm/download/?section=windows) создайте новый файл `Python` (например, `main.py`).

Напишите ваш первый код, например, программу, которая выводит "`Hello, World`!" на экран:

```python
print("Hello, World!")
```
Для отслеживания изменений и коммитов в `GitHub`, откройте [GitHub Desktop](https://github.com/apps/desktop) и увидите изменения, которые вы сделали в проекте.

Добавьте эти изменения и создайте коммит, описав, что было сделано (например, "Добавлен первый скрипт").

Нажмите "`Commit to master`" и "`Push`" для отправки изменений на `GitHub`.

Теперь вы можете делиться своим кодом с друзьями и коллегами, а также отслеживать изменения в проекте.

Теперь у вас есть все необходимые инструменты для эффективной работы: удобная среда разработки, контроль версий с `GitHub` и возможность работать над проектами совместно.

Не забывайте использовать возможности `GitHub` для управления проектами и делиться своим кодом с миром. Удачи в разработке на `Python`! 🚀

Теперь G`itHub Desktop` можно использовать полноценно, так как `Git` будет установлен перед его использованием.
