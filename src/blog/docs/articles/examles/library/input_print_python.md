# Основы Python: функции input() и print()

Когда вы начинаете изучать `Python`, вам обязательно нужно освоить две основные функции: `input()` и `print()`. Они помогут вам взаимодействовать с пользователем и выводить информацию на экран. Давайте рассмотрим их поподробнее!

## 1. Функция print(): вывод информации на экран 🖥️
Функция `print()` используется для вывода текста или значений переменных на экран. Это один из самых простых способов взаимодействовать с программой.


```python
print("Привет, мир!")
```
Этот код выведет:

```
Привет, мир!
```
Вы можете выводить не только строки, но и другие типы данных, такие как числа:

```python
age = 25
print("Мне", age, "лет.")
```
Результат:

```
Мне 25 лет.
```
Если вы хотите вывести несколько элементов в одной строке, `print()` автоматически разделяет их пробелами. Однако можно настроить разделитель, передав параметр `sep`:

```python
print("Python", "is", "awesome", sep="-")
```
Результат:

```csharp
Python-is-awesome
```
## 2. Функция input(): ввод данных от пользователя ⌨️
Функция `input()` позволяет вашему коду получать данные от пользователя. Когда вы используете `input()`, программа приостанавливается и ожидает, пока пользователь введет данные.


```python
name = input("Как вас зовут? ")
print("Привет, " + name + "!")
```
Когда программа выполняется, она сначала попросит пользователя ввести имя, а затем выведет приветственное сообщение.

```
Как вас зовут? Анна
Привет, Анна!
```
## 3. Преобразование введенных данных 🔄
Данные, полученные через `input()`, всегда представляют собой строки. Иногда нужно преобразовать эти данные в другой тип, например, в число.

```python
age = input("Сколько вам лет? ")
age = int(age)  # Преобразуем строку в целое число
print("Через год вам будет", age + 1, "лет.")
```
Если пользователь введет "`25`", программа выведет:

```
Через год вам будет 26 лет.
```

## 4. Важные моменты:
Ввод через `input()` всегда воспринимается как строка. Если вы хотите работать с числами, не забудьте преобразовать строку в нужный тип данных (например, с помощью `int()` или `float()`).
Функция `print()` выводит данные в консоль и завершает строку с новой строки (по умолчанию). Если нужно вывести несколько значений в одной строке, используйте параметр `end`:

```python
print("Привет", end=" ")
print("мир!")
```

Результат:

```
Привет мир!
```

## Заключение 🎯
Теперь вы знаете, как использовать основные функции для взаимодействия с пользователем в `Python` — `input()` и `print()`. Они позволят вам создавать более динамичные программы, которые будут запрашивать информацию у пользователя и выводить результаты. Эти основы пригодятся на протяжении всего пути программирования на `Python`, так что не забывайте про них! 😎
