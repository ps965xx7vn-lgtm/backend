## Основные команды командной строки
Командная строка — это мощный инструмент для управления компьютером с помощью текстовых команд. Она может показаться сложной, но освоив несколько базовых команд, вы сможете эффективно работать с файлами и папками.

## 📌 Как открыть командную строку:
Windows: нажмите `Win` + `R`, введите `cmd` и нажмите `Enter`.
Linux/Mac: откройте терминал (`Ctrl` + `Alt` + `T` на большинстве систем `Linux`).
🚀 Основные команды:
📂 Просмотр текущего каталога:
```bash
pwd  # Linux/Mac
cd   # Windows
```
Команда показывает, в какой папке вы сейчас находитесь.

## 🔄 Переход между папками:
```bash
cd название_папки  # Переход в папку
cd ..              # Переход на уровень выше
```

```bash
cd Documents  # Переход в папку "Documents"
```
## 📋 Просмотр содержимого папки:
```bash
ls     # Linux/Mac
dir    # Windows
```
Выводит список файлов и папок в текущей директории.

## 📁 Создание новой папки:
```bash
mkdir название_папки
```

```bash
mkdir Projects  # Создаёт папку "Projects"
```
## 📝 Создание нового файла:
```bash
touch имя_файла     # Linux/Mac
echo. > имя_файла   # Windows
```

```bash
touch newfile.txt  # Создаёт текстовый файл
```
## Удаление файла или папки:
```bash
rm имя_файла          # Удаляет файл (Linux/Mac)
del имя_файла         # Удаляет файл (Windows)
rm -r название_папки  # Удаляет папку с содержимым
```
## ⚠️ Будьте осторожны: удаление невозможно отменить!

## 🐱 Команда cat: просмотр содержимого файлов
Команда `cat` позволяет вывести содержимое текстового файла на экран.

```bash
cat имя_файла
```

```bash
cat newfile.txt  # Показывает содержимое файла "newfile.txt"
```
## ✍️ Текстовые редакторы в командной строке:
В командной строке вы можете редактировать текстовые файлы прямо из терминала. Самые популярные текстовые редакторы:

- `nano` — простой и удобный редактор для новичков.
```bash
nano имя_файла
```
- `vim` — более мощный редактор для опытных пользователей.
```bash
vim имя_файла
```
---

## 🎯 Почему это важно:
Командная строка — это не только инструмент, но и основа для работы с системами управления версиями (например, `Git`) и автоматизации задач. Освоив базовые команды, вы сможете быстрее и эффективнее управлять проектами! 🚀
