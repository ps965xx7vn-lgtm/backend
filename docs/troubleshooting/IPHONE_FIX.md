📱 БЫСТРАЯ ИНСТРУКЦИЯ ДЛЯ iPhone
====================================

## ШАГ 1: Смени DNS (2 минуты)

1. **Настройки** → **Wi-Fi**
2. Кликни **(i)** рядом с твоей сетью
3. **Настроить DNS** → **Вручную**
4. Удали все, добавь:
   ```
   1.1.1.1
   1.0.0.1
   8.8.8.8
   ```
5. **Сохранить**

## ШАГ 2: Очисти кеш Safari (30 секунд)

1. **Настройки** → **Safari**
2. **Очистить историю и данные**
3. **Очистить**

## ШАГ 3: Перезагрузи телефон (1 минута)

1. **Авиарежим ON** → подожди 10 сек → **OFF**
2. Или выключи/включи iPhone

## ШАГ 4: Открой сайт

1. Safari → **https://pylandschool.com**
2. Если не работает → попробуй **режим инкогнито**

## ⚡ ЕСЛИ СРОЧНО НУЖЕН VPN:

Установи **1.1.1.1: Faster Internet** из App Store
→ Включи тумблер → Готово!
