## 🖥️ Очистка кеша браузера на Mac

### Chrome / Brave / Edge:
1. **Жесткая перезагрузка:** `Cmd + Shift + R`
2. **Или открыть DevTools:** `Cmd + Option + I`
3. **Кликнуть правой кнопкой на кнопку обновления** → **"Очистить кеш и жесткая перезагрузка"**
4. **Или через меню:**
   - Зайти в `chrome://settings/clearBrowserData`
   - Выбрать **"Все время"**
   - Отметить:
     - ✅ Файлы cookie и другие данные сайтов
     - ✅ Изображения и другие файлы, сохраненные в кеше
   - Нажать **"Очистить данные"**

### Safari:
1. **Меню → Разработка → Очистить кеш** (или `Cmd + Option + E`)
2. **Или:**
   - Safari → Настройки → Дополнения
   - ✅ Показывать меню "Разработка"
   - Разработка → Очистить кеш
3. **Полная очистка:**
   - Safari → Настройки → Приватность
   - Управление данными веб-сайтов
   - Удалить все для pylandschool.com

### Firefox:
1. `Cmd + Shift + Delete`
2. Выбрать **"Всё"**
3. Отметить все галочки
4. Нажать **"Очистить сейчас"**

### Быстрый способ (все браузеры):
```bash
# Chrome/Brave
rm -rf ~/Library/Caches/Google/Chrome/Default/Cache/*
rm -rf ~/Library/Caches/BraveSoftware/Brave-Browser/Default/Cache/*

# Safari
rm -rf ~/Library/Caches/com.apple.Safari/*
```

## 📱 Настройка DNS на iPhone/iPad

### Способ 1: Через Wi-Fi настройки
1. Открыть **Настройки** → **Wi-Fi**
2. Нажать на **(i)** рядом с подключенной сетью
3. Прокрутить вниз до раздела **"DNS"**
4. Нажать **"Настроить DNS"**
5. Выбрать **"Вручную"**
6. Удалить все существующие DNS серверы
7. Добавить новые:
   - **1.1.1.1** (Cloudflare - основной)
   - **1.0.0.1** (Cloudflare - резервный)
   - **8.8.8.8** (Google - резервный)
8. Нажать **"Сохранить"**

### Способ 2: Через приложение 1.1.1.1
1. Установить **1.1.1.1: Faster Internet** из App Store
2. Открыть приложение
3. Включить тумблер
4. Это автоматически настроит Cloudflare DNS

1. Открыть https://pylandschool.com

- Подключиться к серверу вне Грузии
